# Student Management System in C

A simple menu-driven Student Management System built using the C programming language.  
This project demonstrates basic concepts of programming such as structures, arrays, loops, and switch-case statements.



##Features

- Add new student records
- Display all students
- Search student by roll number
- Update student details
- Delete student records
- Menu-driven interface



##echnologies Used

- C Programming Language
- Standard Input/Output ('stdio.h')
- Structures ('struct')
- Arrays
- Loops ('for', 'do-while')
- Switch-case statements





## How to Run

1. Download or clone the repository  
2. Open the file in any C compiler (Code::Blocks / VS Code / GCC)  
3. Compile the program  
4. Run the executable
